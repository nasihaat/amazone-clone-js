🛒 Amazon Clone (JavaScript)

A modern Amazon-inspired e-commerce website built with HTML, CSS, and Vanilla JavaScript. Features dynamic product listings, cart functionality with LocalStorage, and responsive design for desktop and mobile.

Inspired by a tutorial from SuperSimpleDev, with personal enhancements and styling applied.

🚀 Features
Dynamic product display
Search products by name
Add/remove items in the cart
Real-time total price calculation
Persistent cart using LocalStorage
Responsive design for all devices
🛠️ Tech Stack
HTML5 – Layout
CSS3 – Styling
JavaScript – DOM manipulation & logic
