# 🛒 Amazon Clone (JavaScript)

A fully functional Amazon-inspired e-commerce website built using **JavaScript**, designed to simulate a real-world online shopping experience. This project demonstrates core frontend development skills including dynamic UI updates, state management, and user interaction handling.

---

## 🚀 Features

* 🏠 Homepage with product listings
* 🔍 Product search functionality
* 🛍️ Add to cart / remove from cart
* 📦 Cart page with total price calculation
* ⭐ Product ratings display
* 🖼️ Dynamic product rendering using JavaScript
* 💾 LocalStorage support (cart persistence)
* 📱 Responsive design

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling
* **JavaScript (Vanilla JS)** – Functionality

---



## ⚙️ Installation & Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/amazon-clone.git
   ```

2. Navigate to the project folder:

   ```bash
   cd amazon-clone
   ```

3. Open `index.html` in your browser.

---

## 💡 How It Works

* Products are stored in a JavaScript file (`products.js`)
* The homepage dynamically renders products using DOM manipulation
* Cart functionality uses **LocalStorage** to persist data
* Users can add/remove items, and the total updates automatically
